# ✨ Features

- 🎭 Browse all Rick & Morty characters
- 🔍 Search characters instantly
- 📖 Character Details Screen
- 🖼 High Quality Character Images
- ⚡ Fast & Responsive UI
- 📱 Adaptive Layout
- 💜 Clean Architecture
- 🧩 Cubit State Management
- 🌐 REST API Integration
- 🚀 Smooth Navigation

# 🛠 Tech Stack

- Flutter
- Dart
- Cubit
- Dio
- REST API
- Clean Architecture
- Dependency Injection
- Cached Network Image
